08_sass함수문법(반복문과 배열)

1. @for문

- @for 키워드는 스타일을 반복적으로 출력하는데 사용한다

- js의 for문과 유사하다.

    ** 문법 1
    through : 종료만큼 반복
    @for $변수 from 시작 through 종료 {
      //반복할 내용
    }

    **문법 2
    //to : 종료 전까지 반복
    @for $변수 from 시작 to 종료{
      //반복할 내용
    }

$변수 => #{변수}

#{변수}는 코드 어디든지 변수 값을 사용할 수 있다

사용 방법:
$img: unquote("test01.jpg");
h1{background-image:url('./img/#{$img}');}
