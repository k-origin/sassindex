if문법
- 조건에 따른 결과를 true/false로 나눠서 처리하는 삼항 연산자와 비슷한 문법
조건식이 true면 '참', 그렇지 않으면 '거짓'으로 반환
    ** 문법
    if(조건식, 참표현식, 거짓표현식);

    조건식?참표현식:거짓표현식;

자바스크립트의 if문과 비슷하게 사용

@else를 붙여 false일 경우 실행, 다중 if문도 실행할 수 있다

    **문법
    1:
      @if(조건){
        //참표현식
      }

    2:
      @if(조건식){
        //참표현식
      }@else{
        //거짓표현식
      }

    3:
      @if(조건식1){
        //조건식1이 참일 때 표현식
      }@else if(조건식2){
        //조건식2가 참일 때 표현식
      }@else{
        //위 조건식 모두 거짓일 때 표현식
      }
